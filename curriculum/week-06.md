## Hands-on

Build a tiny network that learns something simple, such as:

```text
Celsius → Fahrenheit
```

The objective is not model quality. It is to see:

- Initial random weights
- Loss decreasing
- Weights changing
- Saved checkpoint
- Inference from the saved model
